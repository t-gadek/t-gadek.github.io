JSON - Definicja

JSON (JavaScript Object Notation) - jest prostym formatem wymiany danych. Zapis i odczyt danych w tym formacie jest łatwy do opanowania. Jednocześnie, z łatwością odczytują go i generują komputery. Jego definicja opiera się o podzbiór języka programowania JavaScript. JSON jest formatem tekstowym, całkowicie niezależnym od języków programowania.
